# FRP中文客户端
适合服务端管理人员发布客户端的工具，用python构建简洁明了的简体中文界面，简单的配置项，以及中文的日志输出
使用前请看“使用方法”
这只是个业余爱好者自制的工具，bug可能会有点多，只建议不做大规模分发使用的人！！
